# Squabble-Clone

# This is a clone of a website (https://squabble.me/), it allows user to play (very) breif game
# that is identical to Wordle. Most pages just take you through recreations of their web page, slight variations.
# The game is palyed by attempting to guess the hidden word with given feedback from the game on if the letters
# inputed are correct or not, and in the correct location or not. In the settings page the user can change
# the website to a light/dark mode and enable a colorblind feature, that gives an icon to distinguish between
# correct and wrong letters. 

# This is built using HTML, CSS, JavaScript, and jQuery languages/frameworks

# Some comments in the code I have left for future implicaations i wish to make, mainly being responsivness in size and checking other browser supports.
# Only have ran through Edge so far.
